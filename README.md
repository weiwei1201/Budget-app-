# 💰 我的帳本

個人記帳、預算管理、儲蓄目標追蹤 App。

## 功能
- 收支記帳（支出分類、收入分類）
- 多帳戶管理（現金、銀行、線上轉帳）
- 提款與轉帳記錄
- 預算設定與追蹤（每月/每年，自動延用）
- 儲蓄帳戶（目標分配、未分配金額管理）
- 月曆檢視（每日收支點標記）
- 收支圓餅圖分析
- 資料持久儲存於瀏覽器 localStorage

## 本地開發

```bash
npm install
npm run dev
```

## 部署至 Vercel

1. 將此資料夾推上 GitHub Repository
2. 前往 [vercel.com](https://vercel.com) 用 GitHub 登入
3. 點「New Project」→ 選此 Repository → 點「Deploy」
4. 完成！取得 `xxx.vercel.app` 網址

## 手機加入主畫面（PWA）

部署後用手機瀏覽器開啟網址，點「加入主畫面」即可像 App 使用。
